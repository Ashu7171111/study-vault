# Study Vault

A modern and beautifully designed study management web application where students can organize:

- 📚 Subjects  
- 🗂️ Topics  
- 📝 Notes  
- 📄 PDF uploads  

Built with:
- HTML, CSS, JavaScript  
- Supabase (Auth + Database + Storage)  
- Netlify (Hosting)

---

## 🚀 Features

### 🔐 Authentication  
Secure login, signup, logout using Supabase Auth.

### 📚 Subjects  
Create, update, delete subjects with pastel card themes.

### 🗂 Topics  
Expandable topic cards with subtopics, notes and PDF upload options.

### 📝 Notes  
Rich text notes stored safely inside Supabase.

### 📄 PDF Management  
Upload, preview and store PDFs for each topic.

### 🎨 Neon Glow UI (Custom Designed)  
A premium futuristic interface with smooth animations and glass effects.
